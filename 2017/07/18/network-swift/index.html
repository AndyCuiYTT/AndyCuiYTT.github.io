添加网络数据缓存类

简述

  • 网络请求缓存处理
  • 将网络请求结果保存在本地，发起网路请求时先检测缓存区是否有缓存数据并判断是否超出缓存有效时间，如果数据有效则加载缓存区数据，否则加载网络数据.
  • 数据缓存采用 SQLite 存储，采用 FMDB 库.
  • 缓存数据表表数据有 key,value,date 三个字段. key: 网络请求参数 MD5加密数据. value:网络请求数据. date: 数据有效时间
  • 添加计时器,定时清除无效数据.
